# CyberGuardian

**CyberGuardian** es una plataforma web educativa orientada a la concienciación sobre ciberseguridad, protección digital y uso responsable de la tecnología.

El objetivo principal del proyecto es ofrecer información clara, accesible y visualmente atractiva sobre riesgos digitales actuales, herramientas de control parental, buenas prácticas de seguridad y noticias relacionadas con el entorno digital.

---

## Descripción del proyecto

CyberGuardian nace como una web informativa enfocada en ayudar a familias, estudiantes y usuarios en general a comprender mejor los peligros y responsabilidades del mundo digital.

La plataforma combina contenido educativo con un diseño moderno, responsive y fácil de navegar. Su enfoque principal no es prohibir el uso de la tecnología, sino enseñar a utilizarla de forma segura, crítica y responsable.

---

## Características principales

- Página de inicio con presentación del proyecto.
- Secciones educativas sobre riesgos digitales.
- Guías sobre control parental en Android e iOS.
- Apartado de actualidad con noticias y temas relevantes.
- Página de contacto.
- Diseño responsive adaptado a ordenador, tablet y móvil.
- Navegación común entre páginas mediante componentes reutilizables.
- Estilo visual moderno con Tailwind CSS.
- Uso de iconos y tipografías personalizadas.

---

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **Tailwind CSS**
- **JavaScript**
- **Google Fonts**
- **Material Symbols**
- **Firebase** para funcionalidades relacionadas con contacto o envío de información
- **GitHub** para control de versiones
